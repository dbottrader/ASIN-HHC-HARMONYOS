# Vault

The Vault preserves canonical project identity, memory structures, constitutional principles, authorship boundaries, registries, and provenance anchors.

## Public Vault contents

- public project identity and terminology;
- constitutional governance rules;
- artifact registry and manifest pointers;
- public authorship and collaboration boundaries;
- recovery and migration indexes;
- redacted summaries of protected records.

## Protected Vault contents

Sensitive legal, health, family, financial, credential, and private-contact material is not stored in this public repository.

The public Vault may record that a protected artifact exists, together with a non-sensitive identifier and hash, without exposing the artifact itself.
