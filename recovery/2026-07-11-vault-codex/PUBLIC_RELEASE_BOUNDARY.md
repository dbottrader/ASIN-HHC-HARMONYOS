# Public Vault/Codex Release Boundary

This public package preserves the ASIN-HHC / CP8 / HarmonyOS project lineage without publishing unrelated sensitive material.

## Include

- authored technical specifications;
- source code and runnable interfaces;
- public symbolic/codex artifacts;
- machine-readable manifests;
- provenance records;
- public collaboration summaries;
- public research notes;
- redacted legal/IP indexes that do not expose protected details.

## Exclude

- API keys, passwords, access tokens, seed phrases, private keys, or certificates;
- personal medical records or disability records;
- custody, family-law, or child-related private records;
- addresses, phone numbers, birth dates, government identifiers, and financial account data;
- private email/contact lists;
- raw private chat exports;
- files lacking redistribution permission;
- unsupported claims presented as independently validated facts.

## Publishing rule

A file must be reviewed for secrets, privacy, copyright, and claim boundaries before public promotion.

A manifest entry may point to a protected artifact without publishing its contents.

**No receipt means no promotion.**
